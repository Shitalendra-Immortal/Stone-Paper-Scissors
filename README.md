# Stone Paper Scissors - Python Project

A simple, menu-driven Stone-Paper-Scissors game built with Python. This project demonstrates basic game logic, random module usage, and professional Python package architecture.

## 🚀 Features
- **Infinite Loop Menu:** Stay in the game until you choose to exit.
- **Package Architecture:** Logic is separated from the main execution file for better organization.
- **Input Cleaning:** Handles uppercase letters from the user.

## 📁 Project Structure
- `main.py`: The entry point of the application (handles the menu).
- `game_logic/`: A Python package containing the core game engine.
  - `__init__.py`: Package initializer.
  - `engine.py`: Contains the game rules and random result generation.

## 🛠️ How to Run
1. Ensure you have Python installed.
2. Clone this repository.
3. Open your terminal in the project folder.
4. Run the following command:
